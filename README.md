<p>
  駆け抜ける疾走感が特徴の、2Dベルトスクロールランゲームです。
</p>
<p>
  ハイウェイを猛スピードで走り、邪魔をするオブジェクトや追手は
  目にも止まらぬ居合いで斬り伏せます。走り続けるにつれて敵の攻撃も多才になり、
  居合いもよりパワフルになっていきます。
</p>
<p>
  全てを斬り捨て、ひたすらに駆けよ！
